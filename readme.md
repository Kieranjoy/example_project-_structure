# example_project_structure

- This mini project was built to understand how a project in python would typically be structured and utilised in the process.

- Within this build I created some functions and tests to better understand python and how it works creating
simple greeting functions as well as a multiplication functions that takes in a csv number list from the user
and returns the product.

- Aside from this testing file, I also created a script that pulls the characters from the Rick and Morty API
and their respective stats. I wrote this to learn and understand how API calls work in python, what considerations
would be made when calling an API and how to navigate the raw data that is pulled in.
